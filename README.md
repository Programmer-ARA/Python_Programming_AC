## 🚀 How to Run

### Prerequisites
- Python 3.8 or higher — [Download here](https://www.python.org/downloads/)
- No external libraries needed

### Clone & Run

```bash
git clone https://github.com/Programmer-ARA/Python_Programming_AC.git
cd Python_Programming_AC

# Example — run any file
python "Lecture_05 (Loops)/your_file.py"
```
